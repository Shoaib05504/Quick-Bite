# Security Policy

## Supported Versions

We actively support security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please do **NOT** report it publicly via a GitHub issue or pull request. Instead, report it privately to protect users and the integrity of the application.

Please follow these steps:
1. Email your findings to the repository owner or contact listed in [README.md](./README.md).
2. Provide a detailed description of the vulnerability, including:
   - Steps to reproduce the issue.
   - Potential impact.
   - Any suggested remediations or fixes.
3. We will acknowledge receipt of your report within 48 hours and work on a fix in a timely manner.

Thank you for helping keep this project secure!
